belita-shop.by

1) Выбор определенного типа товара.	
	Шаги:
	а) С главной страницы перейти в каталог "Лицо";
	б) В каталоге выбрать категорию "Кремы для лица".
Ожидаемый результат: на странице будут отображены только кремы для лица.

2) Сортировка товаров заданной категории.	
	Шаги:
	а) С главной страницы перейти в каталог "Лицо";
	б) В каталоге выбрать категорию "Кремы для лица".
	в) В селекторе сортировки выбираем "Сначала хиты".
Ожидаемый результат: на странице сначала будут отображены товары-хиты.

3) Поиск товара по названию.	
	Шаги:
	а) В окне поиска ввести "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Нажимаем на кнопку поиска.
Ожидаемый результат: на странице будет отображен товар с этим названием.

4) Добавление заказа в корзину.	+
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Нажимаем на кнопку "В корзину".
Ожидаемый результат: на кнопке появится надпись "Уже в корзине".

5) Добавление нескольких одинаковых товаров в корзину.	
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Увеличиваем количество товаров на 2.
	в) Нажимаем на кнопку "В корзину".
	г) Наводим на кнопку "Корзина".
Ожидаемый результат: количество товаров в корзине будет равным 3.

6) Добавление разных товаров в корзину.		
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Добавить в корзину;
	в) Перейти на вкладку товара "Крем для лица Безупречная кожа";
	г) Добавить в корзину;
	д) Перейти в корзину.
Ожидаемый результат: в корзине будут находиться два этих товара в количестве, указанном при добавлении их в корзину.

7) Удаление товара.	
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Нажимаем на кнопку "В корзину";
	в) Нажимаем на кнопку "Корзина";
	г) Нажимаем на крестик рядом с информацией о товаре.
Ожидаемый результат: товар будет удалён, стоимость заказа изменится.

8) Восстановление товара в корзине.	
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Нажимаем на кнопку "В корзину";
	в) Нажимаем на кнопку "Корзина";
	г) Нажимаем на крестик рядом с информацией о товаре;
	д) Нажимаем на кнопку "Восстановить".
Ожидаемый результат: товар вернется в корзину.

9) Добавление большего количества товаров, чем есть на складе	
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Ставим значение количества товаров 100;
	в) Нажимаем на кнопку "В корзину".
Ожидаемый результат: надпись "Уже в корзине" не появится.

10) Ввод несуществующего кода купона.
	Шаги:
	а) Перейти на вкладку товара "Крем для лица ДВОЙНОЙ КОНТРОЛЬ";
	б) Нажимаем на кнопку "В корзину";
	в) Нажимаем на кнопку "Корзина";
	г) Вводим несуществующий код в окно ввода кода;
	д) Нажимаем кнопку "Добавить".
Ожидаемый результат: выведется ошибка "Купон не найден".
